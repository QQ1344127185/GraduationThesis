# Java毕业设计选题
<p>
每一年都有计算机专业的学弟学妹问我，又到了要做毕业设计了，我要怎么去选题，这里我把我的一些想法分享给大家，希望能够对大家有所帮助。

毕业设计，首先是选题，也就是大家说的开题，这是关键。一般来说，一个同学都是独立完成一个课题，当然也有团队开发的，即几个同学共同分工完成一个课题，这个除外。使用java语言做毕业设计，可选性还是比较多的。

下面我列出一些我做过的选题，方便大家作参考。
